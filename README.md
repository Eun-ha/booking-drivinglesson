## 🚀 시작하기

이 프로젝트를 로컬 환경에서 실행하려면 아래의 단계를 따라 주세요.

#### 1. 레포지토리 클론

```bash
git clone https://github.com/Eun-ha/booking-drivinglesson.git
```

#### 2. 프로젝트 디렉토리로 이동

```bash
cd booking-drivinglesson
```

#### 3. 의존성 설치

```bash
npm install
```

#### 4. 프로젝트 실행

```bash
npm run dev

#or

npmr run start
```
